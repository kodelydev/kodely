# Run Kodely Evals

## Get Started

NOTE: This is MacOS only for now!

Clone the Kodely repo:

```sh
git clone https://github.com/KodelyDev/KodelyApp.git
cd kodely
```

Run the setup script:

```sh
cd evals
./scripts/setup.sh
```

Navigate to [localhost:3000](http://localhost:3000/) in your browser.
